Height and Class Calculator


















Feet'and Inches"

Type the measurements exactly how you would see the 'Tip Measurement' in Ike Office when measuring the top of the pole.

Example: A pole's tip measures to the height of 32'9". In the Feet' field, type '32' and in the Inches" field, type '9'.


Width"

Diameter: Typically you would use this mode the width measuring tool in Ike Office to get this value. Type the diameter of the pole in terms of inches.

Example: A pole's width measures to be 1'3". In the Width' field, type '15'.

Circumference: Select this option to put input the circumference directly.


You can check the step by step math for every "Submit" by Right Clicking anywhere on the page > Inspect > Click on Console tab
